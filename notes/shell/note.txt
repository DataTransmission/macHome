REDIRECTING stdout stderr
-------------------------
# exclamation ! forces overwrite even if noclobber is set
$ cmd > file # will fail if file exist
$ cmd >! file # will write stdout to file, and overwriting any existing file
$ cmd >> file # will fail if file doesn't exist
$ cmd >>! file # will append stdout to file, creating the file if it doesn't exist
